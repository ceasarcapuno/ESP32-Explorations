# ESP32 Explorations

A collection of self-contained ESP32 projects, each in its own folder with its own PlatformIO configuration.

## Projects

| Project | Description |
|---------|-------------|
| [LED-Effects-Controller](LED-Effects-Controller/) | Button-driven multi-mode LED controller — Steady, Blink, Breathing (PWM), and Morse code |

## Structure

```
ESP32-Explorations/
├── LED-Effects-Controller/
│   ├── src/
│   │   └── main.cpp
│   ├── platformio.ini
│   └── README.md
├── LICENSE
└── README.md
```

Each project folder is an independent PlatformIO project. Open the subfolder directly in VS Code / PlatformIO IDE to build and upload.
